# Programação Paralela

Repositório com as atividades desenvolvidas na disciplina de **Programação Paralela**.

## Sobre

Este repositório reúne os exercícios, trabalhos e projetos práticos realizados ao longo da matéria, abordando conceitos como paralelismo, concorrência e computação de alto desempenho.

## Tecnologias utilizadas

- C / C++
- OpenMP
- MPI
- POSIX Threads (pthreads)
